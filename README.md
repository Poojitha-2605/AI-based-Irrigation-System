AI-Based Smart Irrigation & Water Budgeting System
Overview

This project is a data-driven smart irrigation platform that predicts the amount of water required for crops based on environmental conditions. It aims to improve water efficiency and support sustainable agriculture using rule-based decision techniques.

Features
Predicts optimal water requirement based on input parameters
Uses rule-based decision logic for irrigation
Supports water budgeting for efficient usage
Simple and user-friendly interface
Works without hardware using simulated (synthetic) data
How It Works
User provides input values:
Soil Moisture
Temperature
Humidity
The system processes the data and applies:
Rule-based logic (predefined thresholds)
It predicts:
Required water amount
Irrigation decision (whether to water or not)
Tracks water usage for daily budgeting
Dataset
Uses synthetic data (artificially generated but realistic)
Simulates real-world environmental conditions
Helps in testing without sensors or hardware
Objectives
Optimize irrigation decisions using data
Reduce water wastage
Improve agricultural efficiency
Provide a low-cost, scalable solution
Results
Accurate prediction of water requirements
Improved irrigation efficiency
Reduced unnecessary water usage
Structured water budgeting
Societal Impact
Promotes water conservation
Supports farmers in decision-making
Encourages sustainable agriculture
Reduces resource wastage
